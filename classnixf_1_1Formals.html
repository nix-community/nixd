Detailed Description

Lambda formal arguments.

Things to check:

  1. Ellipsis can only occur at the end of the formals. { ..., pkgs } -> { pkgs, ... }
  2. Ellipsis can only occur once. { b, ..., a, ... } -> { a, ... }
