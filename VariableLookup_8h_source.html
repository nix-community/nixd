25/// \brief Represents a definition

187 /// Thus we need to store AST -> Definition
188 /// There are many pointers on AST, the convention is:
189 ///
190 /// 1. attrname "key" syntax is recorded.

192 /// 2. "with" keyword is recorded.
193 /// 3. Lambda arguments, record its identifier.
